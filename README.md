# Cisco Packet Tracer Labs

A collection of Cisco Packet Tracer labs I built while learning LAN networking concepts.

## Labs

### VLAN (`vlan/`)
- **native-vlan.pkt** — Configuring and testing native VLAN behaviour on trunk ports
- **vtp.pkt** — Basic VTP (VLAN Trunking Protocol) setup and propagation

### VTP (`vtp/`)
- **vtp-time-bomb.pkt** — Exploring the VTP time bomb attack scenario

### ACL (`acl/`)
- **acl.pkt** — Creating and applying Access Control Lists to filter traffic

### DHCP (`dhcp/`)
- **dhcp.pkt** — Setting up and testing DHCP for automatic IP assignment

### EtherChannel (`etherchannel/`)
- **etherchannel.pkt** — Configuring EtherChannel for link aggregation

## Requirements

- [Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer) (free download)

## How to Use

1. Download or clone this repo
2. Open any `.pkt` file in Packet Tracer
3. Explore the topology and configurations
